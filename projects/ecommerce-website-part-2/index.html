প্রোজেক্ট সম্পর্কে#

এই প্রোজেক্টটি একটি সম্পূর্ণ ই-কমার্স সমাধান যেখানে রয়েছে:

মূল ফিচারসমূহ#

  • ইউজার রেজিস্ট্রেশন ও লগইন
  • প্রোডাক্ট ক্যাটালগ
  • শপিং কার্ট
  • অনলাইন পেমেন্ট (Stripe)
  • অর্ডার ম্যানেজমেন্ট
  • অ্যাডমিন প্যানেল

ব্যবহৃত প্রযুক্তি#

  • Backend: Laravel 10
  • Frontend: Vue.js 3 + Tailwind CSS
  • Database: MySQL
  • Payment: Stripe API
  • Deployment: AWS/DigitalOcean

চ্যালেঞ্জ ও সমাধান#

এই প্রোজেক্টে বিভিন্ন চ্যালেঞ্জের মুখোমুখি হয়েছি:

  1. পেমেন্ট সিস্টেম: Stripe API integration করতে গিয়ে…
  2. পারফর্মেন্স: বড় ডেটাবেসের জন্য query optimization…

স্ক্রিনশট#

কীভাবে চালাবেন#

# Repository clone করুন
git clone https://github.com/AbdulOhab/ecommerce-project.git

# Dependencies install করুন
composer install
npm install

# Environment setup
cp .env.example .env
php artisan key:generate

# Database migrate করুন
php artisan migrate --seed

# চালান
php artisan serve
npm run dev
